# Goals
- Have a simple yet customizable bootloader for better user experince.
- Have both, PURE/popcorn and polyboot systems.
- Implement a boot protocol
- Learn and Have fun!

# Prerequisites
- Cargo
- M4
- Make
- any package that provides dd
- util-linux (for sfdisk)
